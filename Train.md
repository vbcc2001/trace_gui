
## 一、开发环境准备

### 1. 配置好JDK: 要求JDK17

```bash
java -version
```

### 2. 配置好Maven: 要求3.8以上

```bash
 mvn -v
```


## 二、外设准备

### 1. 标签打印机: Zebra品牌，最好300dpi，并安装好驱动。

官网：https://www.zebra.com/cn/zh/support-downloads.html


### 2. 称重机：要求串口，推荐青松柏品牌

http://www.wooleesmart.com/

### 3. 扫描枪：非必须，可通过键盘模拟
